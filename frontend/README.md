# Frontend - Opinia

Interfaz de usuario para el sistema de encuestas Opinia, enfocada en la simplicidad y usabilidad.

### Tecnologías

- React
- Vite
- Pico CSS

### Librerías Principales

- react-router-dom
- @picocss/pico

### Características

- Diseño minimalista con Pico CSS.
- Exportación de datos a CSV.
- Sistema de feedback para el usuario.

### Instrucciones de ejecución

1. Instalar dependencias: `npm install`
2. Ejecutar en modo desarrollo: `npm run dev`
3. Construir para producción: `npm run build`
