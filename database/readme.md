# Base de datos

La base de datos esta alojada en Supabase, en el archivo **schema.sql** se puede ver los scripts de iniciación utilizados.

### Aclaraciones:

- Se utiliza un soft delete para mantener la persistencia de registros.
- Como identificadores unicos para cada registro se utiliza un tipo UUID que viene implementado en el motor de supabase.

### Instrucciones de ejecución

1. Crear un proyecto en Supabase
2. Dirigirse al editor SQL de supabase
3. pegar y ejecutar el script
